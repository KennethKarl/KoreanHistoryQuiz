## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 앱 이름 | FitTrack |
| 플랫폼 | Android (Kotlin) |
| 대상 사용자 | 운동을 꾸준히 기록하고 싶은 20~40대 |
| 핵심 목표 | 운동 기록 → 분석 → 동기부여 사이클을 만든다 |
| 출시 목표 | 2025년 Q3 |

---

## 주요 기능

### P0 — 필수 (MVP)

- [ ] **회원가입 / 로그인**
  - 이메일 + 비밀번호
  - Google 소셜 로그인
  - JWT 인증, Refresh Token 자동 갱신

- [ ] **운동 기록 입력**
  - 종목 선택 (달리기, 헬스, 수영 등)
  - 시간·거리·칼로리 입력
  - 날짜·시간 자동 기록

- [ ] **기록 목록 / 상세 조회**
  - 날짜별 운동 히스토리
  - 운동 상세 정보(지도, 경로, 심박수 등)

- [ ] **홈 대시보드**
  - 이번 주 운동 요약 (횟수, 총 시간, 칼로리)
  - 연속 달성 일수 (스트릭)

### P1 — 중요

- [ ] **통계 화면**
  - 주간 / 월간 운동량 차트
  - 종목별 비율

- [ ] **목표 설정 및 달성률**
  - 주간 목표 입력
  - 달성률 시각화

- [ ] **푸시 알림**
  - 운동 리마인더 (FCM)
  - 목표 달성 축하 알림

### P2 — 선택

- [ ] **소셜 공유**
  - 운동 결과 이미지 생성 → SNS 공유

- [ ] **친구 목록 / 랭킹**
  - 친구와 주간 운동량 비교

## 화면 목록

| 화면 | 설명 |
|------|------|
| 스플래시 | 앱 로딩 |
| 온보딩 | 최초 실행 가이드 (3장) |
| 로그인 | 이메일 / 소셜 로그인 |
| 회원가입 | 이메일 + 닉네임 + 비밀번호 |
| 홈 | 대시보드 (오늘 요약, 스트릭) |
| 기록 입력 | 운동 종목·시간·거리 입력 |
| 기록 목록 | 날짜별 운동 히스토리 |
| 기록 상세 | 운동 상세 (지도, 심박수) |
| 통계 | 차트, 종목별 분석 |
| 목표 설정 | 주간 목표 입력 |
| 내 정보 | 프로필, 알림 설정, 로그아웃 |

## 비기능 요구사항

| 항목 | 기준 |
|------|------|
| 콜드 스타트 시간 | ≤ 2초 |
| API 응답 후 렌더링 | ≤ 100ms |
| 최소 지원 OS | Android 8.0 (API 26) |
| 메모리 사용량 | ≤ 150MB (일반 사용 시) |
| 크래시율 | ≤ 0.1% |
| 오프라인 지원 | 기록 입력은 오프라인 가능, 동기화는 온라인 시 자동 |
