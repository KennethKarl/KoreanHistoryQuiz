# Claude Agent 프로젝트 설정

이 프로젝트는 Claude Agent 멀티세션 구조로 운영됩니다.

## 가이드 위치
- `guides/guide-common.md` — 공통 가이드
- `guides/guide-session-{0~7}-*.md` — 세션별 가이드

## 핵심 규칙
1. 작업 전 `.session-status.json`과 `memos/claude-mistakes.md`를 반드시 읽어라.
2. 아키텍처 참조는 `tasks/architecture.md`(원본)만 사용하라.
3. 코드 작성 전 개발자 승인을 받아라.
